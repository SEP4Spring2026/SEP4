# Non-Functional Requirements
<!-- Rebeca, Soma, Piotr -->

## Maintainability

1. The system must use Git with a central repository (GitHub or GitLab) for version control.

2. The project must maintain a clear and well-documented commit history.

3. The development process must follow SCRUM and AUP methodologies.

## Reliability and Testability

4. The system must include unit tests for parts of the application.

5. The project must implement automated regression testing using GitHub/GitLab CI features.

## Deployability and Scalability

6. The system mut be deployed using containers.

7. The backend must be hosted on a public cloud provider.

8. Parts of the backend must run as serverless workloads.

## Security

9. Communication between the IoT device and the cloud should support encrypted connections.

10. Backend endpoints should support authentication using JSON Web Tokens (JWT).

## Usability

11. The web application must provide a responsive user interface.

12. The interface must support screen widths of 576px, 768px, and 1200px.
